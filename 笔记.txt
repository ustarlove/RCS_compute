从 l=−1 开始是为了递推统一：
  代入 h_0 = (sin x + j cos x)/x 和 h_1 = (sin x + j cos x)/x² + (-cos x + j sin x)/x，化简即得 h_{-1} = (cos x - j sin
  x)/x = e^{-jx}/x

  为什么选多余带宽公式

  1. 精度可控 — log₁₀(1/ε) 项让使用者可以根据需要的精度调整 L。代码中 ε = 1e-3 对应 log₁₀(1/ε) =
  3.0，如果要求更高精度只需改这个参数。
  2. 更标准的工程参考 — Chew et al. 的 FMM 教材/论文是计算电磁学中 FMM 实现最常引用的文献。
  3. 实际上结果差不多 — 以 box_size = 0.3λ 为例：
    - a ≈ 0.26λ, ka ≈ 1.63
    - 代码公式: L = 1.63 + 5.4 = 7.03 → L ≈ 7
    - 半经验公式 (d = 2a, kd = 3.26): L = 3.26 + 2ln(3.26+π) = 3.26 + 3.71 = 6.97 → L ≈ 7

  两者接近，但多余带宽公式在高频端 (ka 大时) 增长更保守（ka^{1/3} 项增长慢于 ln(ka)）
